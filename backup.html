The Foundling Wheel Archive

The Foundling Wheel is a digital platform supported by a knowledge repository which fully automates the processing of information.

The Archivist from the Municipal Archive of Fafe provided transcripts in a digital format. Each document is spellchecked and an autocomplete system restores unreadable sections. Then a named entity recognition, relation extraction and entity linking components extract all the important pieces of information.

The resulting data is structured and stored accordingly to an ontology, which is also responsible for deriving additional knowledge via the usage of reasoners.

This video showcases the application in action, by processing a transcript via the aforementioned processing pipeline.

Main technologies used:

  • Python
  • Spacy
  • OwlReady 2
  • HermiT
  • Hunspell
  • Fuzzywuzzy
  • Django
  • Bootstrap

Spickles

Spickles is a mobile phone application that assists their users choosing a movie to watch. The key difference between Spickles and generic movie recommendation systems is this application provides recommendations not only for a specific user but also assists the decision process for an arbitrary sized group.

By creating a group, the application takes in consideration potential filters added and the preferences of each member. Then the recommendation engine creates a targeted and small pool of movies.

Finally each member is able to play a swipe left / right game anonymously over the generated pool of movies, where the end, a consensus shared movie is chosen to be watch.

I was a member of the development team of Spickles, in total the team was composed of 9 members.

Main technologies used:

  • JavaScript
  • Python
  • React Native
  • Django
  • Node JS
  • Scikit-Learn
  • Scipy
